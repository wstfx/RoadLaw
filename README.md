# 路律：交通法规与科目一学习系统

这是一个以 C1/C2 科目一为主、同时帮助建立中国大陆道路交通法规全景的本地学习网页。

## 使用

直接打开 `index.html`，或在本目录运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。学习勾选进度保存在浏览器 `localStorage` 中，不会上传。

## 学习地图

1. 基础法理与道路秩序
2. 交通信号与道路设施
3. 道路通行规则
4. 驾驶证申领与使用
5. 机动车与登记管理
6. 违法、处罚与记分
7. 事故处理与应急避险
8. 安全文明与综合责任

推荐顺序是：先看“法规体系”，再按大纲逐项勾选；每完成一个模块，用“知识校验”确认是否真的理解规则。

侧边栏“课程中心”目前包含：

- 模块 01：7 节语义化讲解、35 道逐节练习。
- 模块 02：8 节详细课程、48 道逐节练习，包含信号灯、标志、标线、交警手势、车辆灯光，以及“电动车／电瓶车”等易混概念。

课程答错后会显示解释；章节完成状态会自动同步回理论大纲。课程中心使用数据目录生成，后续模块可以继续按相同结构添加。

## 内容边界

- 法规基线核对至 2026 年 7 月。
- 驾驶证规则采用公安部第 172 号令修改后的版本（2025 年 1 月 1 日起施行）。
- 记分规则采用公安部第 163 号令，分值档位为 12、9、6、3、1 分。
- 机动车登记采用公安部第 164 号令。
- 考试框架参考《机动车驾驶证申领和使用规定》与 GA 1026—2022。
- 示例题为学习用途，不复制或替代正式题库；各地临考通知以所在地车管所和“交管 12123”为准。
